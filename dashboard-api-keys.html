Quick Start Guide

Use our REST API to integrate Leadbunker into your applications

Authentication

Include your API key in the Authorization header:

curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.leadbunker.com/v1/extract

Start Extraction

curl -X POST https://api.leadbunker.com/v1/extract \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "type": "full"}'

Get Task Status

curl https://api.leadbunker.com/v1/tasks/TASK_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
