Logo Guidelines

  • File formats: PNG (with transparency) or SVG recommended
  • File size: Maximum 500KB per logo
  • Dimensions: Minimum 200x60px, recommended 400x120px
  • Light logo: Should work on white/light backgrounds
  • Dark logo: Should be white or light colored for dark backgrounds
  • Favicon: Square format (32x32 or 64x64 pixels)
  • Testing: Always preview logos on both light and dark backgrounds
